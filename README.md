# GUI工具使用说明

## 🎨 界面介绍

这是一个基于tkinter的图形界面工具，提供便捷的加密和解密功能。

## 🚀 启动方法

```bash
python encrypt_decrypt_gui.py
```

## 📋 功能说明

### 加密标签页

1. **输入JSON数据**
   - 在输入框中输入要加密的JSON数据
   - 支持自动修复格式（如 `{status: 1}` 会自动修复为 `{"status": 1}`）

2. **加密按钮**
   - 点击"加密"按钮执行加密操作
   - 加密结果会显示在下方的输出框中

3. **加载示例**
   - 点击"加载示例"按钮可以快速加载预设的示例数据

4. **复制结果**
   - 点击"复制结果"按钮可以将加密后的字符串复制到剪贴板

### 解密标签页

1. **输入加密字符串**
   - 在输入框中输入要解密的Base64字符串

2. **解密按钮**
   - 点击"解密"按钮执行解密操作
   - 解密结果会以格式化的JSON显示在下方的输出框中

3. **复制结果**
   - 点击"复制结果"按钮可以将解密后的JSON复制到剪贴板

## 💡 使用示例

### 示例1: 加密活动状态

1. 切换到"加密"标签页
2. 在输入框中输入: `{"status": 1}`
3. 点击"加密"按钮
4. 查看加密结果
5. 点击"复制结果"复制到剪贴板

### 示例2: 解密数据

1. 切换到"解密"标签页
2. 在输入框中输入加密字符串: `hfGzD3WWYdsboWvSsZ8FEg==`
3. 点击"解密"按钮
4. 查看解密后的JSON结果
5. 点击"复制结果"复制到剪贴板

## ⚙️ 功能特点

- ✅ **双标签页设计**: 加密和解密功能分离，界面清晰
- ✅ **自动格式修复**: 自动修复PowerShell引号问题
- ✅ **示例数据**: 快速加载常用示例
- ✅ **一键复制**: 方便复制结果
- ✅ **状态提示**: 实时显示操作状态
- ✅ **自动下载**: 首次使用时自动下载WASM模块
- ✅ **错误提示**: 友好的错误提示信息

## 📝 注意事项

1. **首次使用**: 首次运行时会自动下载WASM模块（需要网络连接）
2. **Node.js要求**: 需要安装Node.js（v14+）
3. **JSON格式**: 输入必须是有效的JSON格式
4. **网络连接**: 首次下载WASM模块需要网络连接

## 🔧 故障排除

### 问题1: "未找到Node.js"

**解决**: 安装Node.js: https://nodejs.org/

### 问题2: "下载WASM模块失败"

**解决**: 
- 检查网络连接
- 手动下载WASM模块到当前目录

### 问题3: "加密/解密失败"

**解决**:
- 检查输入格式是否正确
- 查看错误提示信息
- 确保WASM模块已正确下载

## 🎯 快捷键

- `Ctrl+C`: 复制选中内容
- `Ctrl+V`: 粘贴内容
- `Ctrl+A`: 全选
